    Numerical Integration¶

    1. Apa itu Integrasi Numerik ?¶

    ​ Integrasi numerik merupakan algoritma untuk menghitung nilai numerik dari integral tentu dan dengan ekstensi, terkadang integrasi numerik digunakan juga untuk mendeskripsikan solusi numerik dari persamaan diferensial. Integrasi numerik digunakan untuk menghitung solusi aproximat untuk mendefinisikan integral.

    Menghitung Integral tentu : $$ I\left( f\right) =\int ^{b}_{a}f\left( x\right) dx $$ Nilai pendekatan di atas diperoleh dari penyelesaian numerik f fungsi kontinu pada interval a,b.

    2. Metode Interasi Numerik¶

    Metode-MetodeIntegrasi Numerik :

    1. Trapezoid
    2. Romberg
    3. Simpson ⅓
    4. Simpson ⅜

    2.1 Metode Trapezoid¶

    2.1.1 Materi Metode Trapezoid¶

    ​ Metode Trapezoid juga disebut metode trapezium menggunakan pendekatan area di bawah kurva dengan memasang trapezium dengan interval spasi teratur. Pada metode ini, fungsi sebagai dua kaki trapezium, sedangkan interval ditentukan dari tinggi trapezium. Luas trapezium adalah ½ tinggi di kalikan dengan jumlah kedua alasnya, L = ½(t x (a+b)) sehingga :

    \[ \int ^{b}_{a}f\left( x\right) dx=\left( b-a\right) \dfrac{f\left( a\right) +f\left( b\right) }{2} \]

    batas akhir interval - batas awal interval = h, sehingga :

    $$ \int ^{b}_{a}f\left( x\right) dx=h \dfrac{f\left( a\right) +f\left( b\right) }{2} $$ Rumus di atas digunakan untuk 1 partisi, lalu bagaimana jika terdapat 2 partisi atau lebih ?

    \[ T_{2}\left( f\right) =\dfrac{1}{2}h\left[ f\left( x_{0}\right) +f\left( x_{1}\right) \right] +\dfrac{1}{2}h\left[ f\left( x_{1}\right) +f\left( x_{2}\right) \right] \]
    \[ =\dfrac{1}{2}h\left[ f\left( x_{0}\right) +2f\left( x_{1}\right) +f\left( x_{2}\right) \right] \]
    \[ =h\left[ \dfrac{1}{2}( ( f\left( x0\right) +\left( f\left( x_{2}\right) \right) +f\left( x_{1}\right) \right] \]

    Karena Karena x1-x0 = x2-x1, maka h = (b-a)/2, maka : $$ =\dfrac{b-a}{2}\left[ \dfrac{1}{2}( \left( f\left( x_{0}\right) +f\left( x_{2}\right) \right) +f\left( x1\right) \right] $$ Sehingga rumus umum untuk 2 partisi atau lebih, sebagai berikut : $$ T_{n}\left( f\right) =\dfrac{b-a}{n}\left[ \dfrac{1}{2}( \left( f\left( x0\right) +f\left( xn\right) \right) +2\sum ^{n-1}_{i=1}f\left( xi\right) \right] $$

    2.1.2 Algoritma Metode Trapezoid¶

    1. Mendefinisikan fungsi f(x)
    2. Menentukan Interval
    3. Menentukan jumlah pias n
    4. Menentukan nilai h
    5. Melakukan perhitungan dan iterasi

    2.1.3 Code Metode Trapezoid¶

    #mendefinisikan fungsi f(x)
def f(x):
    #contoh fungsi f(x)=x**2
    y=x**2
    return y

#a=batas awal interval
#b=batas akhir interval

def trapezoid(a,b,n):
    h=(b-a)/n #mencari nilai h
    hasilSementara=0.0
    #melakukan perulangan
    for i in range(1,n):
        #mencari xi
        xi=a+i*h
        #menjumlahkan f(xi)
        hasilSementara+=f(xi)
    #menghitung nilai I
    I=h/2*(f(a)+2*hasilSementara+f(b))
    return I
#contoh mencari trapezoid dengan a=0,b=2,n=6
print (trapezoid(0,2,6))

    2.2 Metode Integrasi Romberg¶

    2.2.1 Materi Integrasi Romberg¶

    Integrasi Romberg merupakan gabungan dari metode trapezoid dengan ekstrapolasi Richardson. Dengan metode ini akan didapatkan error yang lebih sedikit sehingga hasil menjadi lebih akurat.

    Table metode romberg :

    R0,0
    R1,0 R1,1
    R2,0 R2,1 R2,2
    . ... ... ...
    Rn,0 Rn,1 Rn,2 ... Rn,n

    Rumus umum metode romberg :

    Perhitungan dimulai dari R1,1 dan R2,1 sehingga didapatkan R2,2. Dengan demikian, berdasarkan table di atas, R pada kedua pangkal panah harus di hitung terlebih dahulu untuk mendapatkan hasil yang dituju. Untuk mendapatkan hasil akhir, dilakukan iterasi sampai pojok bawah kanan table.

    Rumus Integrasi Romberg :

    jika i=1 :

    \[ Rij=\dfrac{h}{2}[ f < x1) +2\sum ^{n-1}_{i=2}f\left( x_{i}\right) +f\left( \times 1\right) ] \]

    jika i>1 :

    \[ Rij=\dfrac{4j-1_{Rij-1-R_{i}-1,j-1}}{4^{j-1}-1} \]

    2.2.2 Algoritma Integrasi Romberg¶

    • Menghitung nilai h
    • Menghitung kolom pertama dari dua baris awalmenggunakan metode trapezoid

    • Menghitung kolom selanjutnya pada baristersebut dengan rumus Romberg untuk i>1

    • Mengulagi perhitungan sampai pada Rn,n

    2.2.3 Code Integrasi Romberg¶

    import numpy as np 
import math
#menentukan f 
f=lambda x: math.exp(x)
#menentukan a dan b
a=0
b=1
def trapezoid(a,b,n):
    h=(b-a)/n 
    hasilSementara=0.0
    for i in range(1,n):
        xi=a+i*h
        hasilSementara+=f(xi)
    I=h/2*(f(a)+2*hasilSementara+f(b))
    return I

I=np.zeros((10,10))
ea=100
n=1
i=1
while ea >0.1:
    n=2**1
    #menghitung menggunakan trapezoid
    I[i+1,1]=trapezoid(a,b,n)
    for k in range (2,i+2,1):
        j=2+i-k
        #metode romberg
        I[j,k]=(4**(k-1)*I[j+1,k-1]-I[j,k-1])/(4**(k-1)-1)

    ea= abs((I[1,i+1]-I[2,i])/I[1,i+1])*100
    et=abs((1.71828-I[1,i+1])/1.7128)*100

    print (i,ea,et,I[1,i+1])
    i+=1

    2.3 Metode Integrasi Simpson ⅓¶

    2.3.1 Materi Integrasi Simpson ⅓¶

    Metode simpson adalah metode yang mencocokan polynomial derajat 2 pada tiga titik data diskrit yang mempunyai jarak yang sama (Ema et al, 2017). Hampiran nilai integrasi yang lebih baik dapat ditingkatkan dengan menggunakan polinom interpolasi berderajat yang lebih tinggi.

    Rumus umum metode simpson ⅓ :

    \[ I=\dfrac{h}{3}\left( f_{0}+4\sum ^{n-1}_{i=1,3,5}fi+2\sum ^{n-2}_{i=2,4,6}fi+fn\right) \]

    2.3.2 Algoritma Integrasi Simpson ⅓¶

    1. Mendefinisaikan fungsi f(x) yang akan diintegrasikan

    2. Menentukan batas bawah (a) dan batas atas (b) interval

    3. Menentukan jumlah pias n (n kelipatan 2)
    4. Menghitung lebar pias h
    5. Melakukan iterasi dan menghitung menggunakan rumus metode simpson ⅓

    2.3.3 Code Integrasi Simpson ⅓¶

    #mendefinisikan fungsi f(x)
def f(x):
    #contoh fungsi f(x)=1/1+x
    y=1/(1+x)
    return y

#a=batas awal interval
#b=batas akhir interval

def Simpson13(a,b,n):
    h=(b-a)/n #mencari nilai h
    hasilgenap=0.0
    hasilganjil=0.0
    #melakukan perulangan
    for i in range(1,n):
        if i%2==1:
            #mencari xi tiap iterasi ganjil
            xi1=a+i*h
            #menjumlahkan f(xi)ganji
            hasilganjil+=f(xi1)
        else :
            #mencari xi tiap iterasi genap
            xi0=a+i*h
            #menjumlahkan f(xi) genap
            hasilgenap+=f(xi0)
    #menghitung nilai I
    I=h/3*(f(a)+4*hasilganjil+2*hasilgenap+f(b))
    return I
#contoh mencari Simpson 1/3 dengan a=0,b=1,n=8
print (Simpson13(0,1,8))

    2.4 Metode Integrasi Simpson ⅜¶

    2.4.1 Materi Intergrasi Simpson ⅜¶

    Metode simpson ⅜ hampir sama denga metode simpson ⅓ , namun pada metode ini mencocokan polynomial derajat 2 pada 4 titik data diskrit yang mempunyai jarak yang sama. Hampiran nilai integrasi yang lebih baik dapat ditingkatkan dengan menggunakan polinom interpolasi berderajat yang lebih tinggi.

    Rumus Integrasi Simpson ⅜ :

    \[ I=\dfrac{3h}{8}\left( f_{0}+3\sum ^{n-1}_{i\neq3,6,9}fi+2\sum ^{n-3}_{i=3,6,9}fi+fn\right) \]

    2.4.2 Algoritma Integrasi Simpson ⅜¶

    1. Mendefinisaikan fungsi f(x) yang akan diintegrasikan

    2. Menentukan batas bawah (a) dan batas atas (b) interval

    3. Menentukan jumlah pias n (n kelipatan 3)
    4. Menghitung lebar pias h
    5. Melakukan iterasi dan menghitung menggunakan rumus metode simpson ⅜

    2.4.3 Code Integrasi Simpson ⅜¶

    #mendefinisikan fungsi f(x)
def f(x):
    #contoh fungsi f(x)=1/1+x
    y=1/(1+x)
    return y

#a=batas awal interval
#b=batas akhir interval

def Simpson3per8(a,b,n):
    h=(b-a)/n #mencari nilai h
    hasil1=0.0
    hasil2=0.0
    #melakukan perulangan
    for i in range(1,n):
        if i%3==0:
            #mencari xi tiap iterasi kelipatan 3
            xi1=a+i*h
            #menjumlahkan f(xi) kelipatan 3
            hasil1+=f(xi1)
        else :
            #mencari xi tiap iterasi bukan kelipatan 3
            xi0=a+i*h
            #menjumlahkan f(xi) bukan kelipatan 3
            hasil2+=f(xi0)
    #menghitung nilai I
    I=(3*h)/8*(f(a)+3*hasil2+2*hasil1+f(b))
    return I
#contoh mencari hasil dari a=0,b=1.125,n=9
print (Simpson3per8(0,1.125,95))
