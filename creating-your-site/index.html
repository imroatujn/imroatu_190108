    Numerical Solution of System of Linear Algebraic Equation¶

    1. Sistem Persamaan Linear¶

    Metode numerik adalah salah satu cabang ilmu matematika yang banyak menyelesaikan permasalahan. Analisis numerik dilakukandengan melakukan pendekatan numerik untuk masalah matematis.

    Sistem Persamaan Linear telah banyak di aplikasikan dalam berbagai bidang pengetahuan.

    Sistem persamaan linear adalah sekumpulan persamaanlinear dengan berbagai variable. Pada sistem persamaan linear terdapat 2 metode yaitu metode langsung dan metode iterative.

    Metode langsung menggunakan metode numerik analitik dan penyelesaiannya dalam Langkah yang terbatas. Sedangkan Metode iteratif menggunakan solusi awal dan konvergen ke nilai yang mendekati solusi eksak dengan menyempurnakan solusi awal secara iteratif. penyelesaian metode iterative berukuran lebih besar dan memiliki nilai koefision yang lebih banyak.

    Berikut merupakan Sistem umum dari persamaan linear m dengan jumlah n yang tidak diketahui : $$ \begin{aligned}a_{11}x_{1}+a_{12}x_{2}+\ldots +a_{1}nx n=b1\end{aligned} $$

    \[ \begin{aligned} a_{21}x_{1}+a22\times 2+\ldots +a2nx\eta =b_{2}\end{aligned} \]
    \[ \begin{aligned} \vdots \\ \end{aligned} \]
    \[ \begin{aligned}am_{1}x_{1}+am_{2}x_{2}+\ldots +amnxn=bm\end{aligned} \]

    2.1 Metode Eliminasi Gauss¶

    2.1.1 Materi Eliminasi Gauss¶

    Metode Gauss merupakan metode langsung yang menggunakan pendekatan analitik, metode penyelesaian menggunakan metode gauss ini dilakukan dengan memperhatikan koefisien dibawah diagonal utamanya, kemudian koefisien tersebut diubah menjadibernilai nol, sehingga persamaan linear ini berbentuk seperti segitiga atas. Metode Eliminasi Gaus dapat diselesaikan dengan metode subtitusi mundur.

    Pada eliminasi gauss dapat digunakan operasi baris elementer untukmendapatkan nilai 0 pada bawah diagonal utama. aturannya sebagai berikut :

    1. Menukar satu persamaan dengan persamaan pada baris lain

    2. Menjumlahkan persamaan dengan persamaan lain

    3. Mengalikan dengan persamaan lain, dengan syarat konstanta tidaksama dengan 0

    Langkah penyelesaian mengunakan metode gauss ada 2 metode :

    1. Eliminasi maju
    2. Subtitusi mundur

    Pada tahap eliminasi dapat dilakukan menggunakan aturan OBE untukmenghasilkan koefisien bernilai 0 di bawah diagonal utama. Pada system terakhir sampai hanya tersisa annxn. sehingga system persamaan atau matrix yang dihasilkan menjadi berbentuksegitiga atas. Setelah didapatkan matrix segitiga atas, maka dilakukansubtitusi yang awalnya dimulai dari annxn pada persamaan terakhir, laludimasukkan ke persamaan di atasnya sampai pada persamaan pertama dan didapatkan hasil yang diinginkan.

    2.1.2 Agoritma Eliminasi Gauss¶

    Langkah-langkah menyelesaikan Sistem Persamaan Linear menggunakan metode Eliminasi Gauss :

    1. Menentukan persamaan yang akan dicari
    2. Melakukan eliminasi maju dengan operasi baris elementer
    3. Melakukan subtitusi mundur
    4. Mendapatkan solusi

    2.1.3 Code Eliminasi Gauss¶

    import numpy as np
import sys

n = int(input('masukkan Besar matrix: '))
a = np.zeros((n,n+1))
b = np.zeros(n)
print('Masukkan koefisien matrix : ')
#memasukkan nilai ke matrix a (augmented matrix)
for i in range(n):
    for j in range(n+1):
        a[i][j] = float(input( 'a['+str(i)+']['+ str(j)+']='))
#melakukan eliminasi maju
for i in range(n):
    #jika terdapat koefision 0, maka sistem memberitahu eror
    if a[i][i] == 0.0:
        sys.exit('terdeteksi penyebut 0!')

    for j in range(i+1, n):
        banding = a[j][i]/a[i][i]

        for k in range(n+1):
            a[j][k] = a[j][k] - banding * a[i][k]

b[n-1] = a[n-1][n]/a[n-1][n-1]

#melakukan subtitusi mundur 
for i in range(n-2,-1,-1):
    b[i] = a[i][n]

    for j in range(i+1,n):
        b[i] = b[i] - a[i][j]*b[j]

    b[i] = b[i]/a[i][i]

print('\nSolusi : ')
for i in range(n):
    print('X%d = %0.2f' %(i,b[i]), end = '\t')

    2.2 Metode Iterasi Jacobi¶

    2.2.1 Materi Iterasi Jacobi¶

    Metode jacobi meupakan metode iterative, yang cocok digunakanuntuk system dengan koefision yang besar. Metode ini melakukan analisis padatitik awal dan maju untuk mendapatkan solusi berikutnya.

    Contoh SPL iterasi jacobi :

    $$ \begin{aligned}a_{11}x+a_{12}y+a_{13}z=b_{1}\ a_{21}x+a_{22}y+a_{23}z=b_{2}\ a_{31}x+a_{31}y+a_{33}z=b_{3}\end{aligned} $$ Aturan iterasi jacobi :

    $$ x_{i}=\dfrac{bi-\sum {i\neq j}aijx}{aii} $$ iterasi metode jacobi akan berakhir jika :

    \[ X_{1}^{n-1}\approx X_{1}^{n},X_{2}^{n-1}\approx X_{2,}^{n}X_{3}^{n-1}\approx X_{3}^{n} \]

    2.2.2 Algoritma Iterasi Jacobi¶

    Langka-langkah melakukan iterasi jacobi sebagai berikut :

    1. Melakukan pendekatan awal
    2. Menentukan nilai toleransi

    3. Menentukan Matriks Persamaan Linear

    4. Menghitung menggunakan aturan jacobi

    5. Melakukan iterasi

    6. Mendapatkan hasil

    2.2.4 Code Iterasi Jacobi¶

    import numpy as np

#vektor A
iterasi=int(input("masukan iterasi yang diinginkan = "))
n = int(input('masukkan jumlah persamaan linear: '))
A = np.zeros((n,n))
#C = list untuk menampung x1 baris 1, x2 baris 2, dan x3 baris 3
C=[]
mat=[[0,0,0]]
#vektor b
b =np.zeros((n))

#memasukkan nilai ke dlam matrix kosong pada matrix a
for i in range(0,n,1):
    for j in range(0,n,1):
        A[i][j] = float(input( 'A['+str(i)+']['+ str(j)+']='))

#MEMASUKKAN NILAI PADA b
for i in range(0,n,1):
     b[i] = float(input( 'b['+str(i)+']='))


#MEMBUAT LIST BARU UNTUK FUNGSI BARU DAN PEMINDAHAN RUAS
for i in range (n):
    for j in range (n):
        if (i==j):
            #memasukkan x1 baris 1, x2 baris 2, dan x3 baris 3 pada list C
            C.append(A[i][j])
            #membuat x1 baris 1, x2 baris 2, dan x3 baris 3 menjadi 0 pada list A
            A[i][j]=0
        else:
            #karena pindah ruas maka tanda juga ikut berubah
            A[i][j]=-A[i][j]
print("A baru = ",A)
print("C =",C)

print ("iterasi ke-0 = ",mat)

#perulangan k untuk melakukan perulangan sesuai iterasi yang diminta
#mencari hasil/solusi sementara per iterasi
for k in range (1,iterasi+1,1):
    #matt = list untuk menampung hasil sementara 
    matt=[]
    for i in range (0,len(mat[k-1]),1):
        hasil=0
        for j in range (0,n,1):
            hasil+=(A[i][j]*mat[k-1][j])
        hasil=(hasil+b[i])/C[i]
        matt.append(hasil)
    #menambah index baru yang berupa list yang berasal dari list matt
    mat.append(matt)
    print("iterasi ke-",k,"=",mat[k])

print("solution = ",mat[iterasi])
print ('mat = ',mat)

    2.3 Metode Iterasi Gauss Seidel¶

    2.3.1 Materi Iterasi Gauss Seidel¶

    Iterasi gaus seidel dapat dikatakan sebagai modifikasi dari metode iterasi jacobi. Persamaan pada metode iterasi ini juga sama dengan metode jacobi, namun metode ini lebih sedikit komputasinya dan lebih cepat dari metode jacobi. Perbedaan antara kedua metode iterasi ini adalah pada pemanfaatan nilai yang telah didapatkan dari setiap iterasi, pada iterasi seidel nilai yang telah didapatkan, digunakan untuk mencari nilai pada iterasi berikutnya.

    2.3.2 Algoritma Iterasi Gauss Seidel¶

    Algoritma iterasi seidel sebagai berikut :

    1. Melakukan pendekatan awal
    2. Menentukannilai toleransi

    3. MenentukanMatriks Persamaan Linear

    4. Menghitungmenggunakan aturan Seidel

    5. Melakukaniterasi
    6. Mendapatkanhasil
